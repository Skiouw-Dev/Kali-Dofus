# Kali ⚔

Gestionnaire de fenêtres multi-comptes pour **Dofus** — bascule entre tes personnages avec des raccourcis clavier, dans une interface sombre style Windows 11.

## 📥 Téléchargement

**[⬇️ Télécharger Kali (Windows)](https://github.com/Skiiouw/Kali-Dofus/releases/latest/download/Kali-Setup.exe)**

Lance l'installateur, c'est tout — Kali se met ensuite **à jour automatiquement** à chaque démarrage.

> ℹ️ Au premier lancement, Windows SmartScreen peut afficher un avertissement
> (application non signée) : clique sur **« Informations complémentaires »**
> puis **« Exécuter quand même »**.

## ✨ Fonctionnalités

- **Détection automatique** des fenêtres Dofus (fiable, par processus — pas de faux positifs)
- **Ordre d'initiative** par glisser-déposer fluide et animé
- **Raccourcis clavier globaux** (fonctionnent même en combat) :
  - Perso suivant / précédent (F1 / F2, configurables)
  - Accès direct Alt+1 à Alt+8 (modificateur réglable, sans conflit avec les barres de sorts)
- **Chrono de session** dans la barre de titre + notification Windows du temps de jeu
- **Zone de notification** : Kali se range à côté de l'horloge quand tu le réduis
- **Une seule instance** : impossible de le lancer deux fois par erreur
- **Mises à jour automatiques** via GitHub — installe une fois, à jour pour toujours

## 📖 Utilisation

1. Lance tes comptes Dofus
2. Ouvre Kali : tes personnages apparaissent automatiquement
3. Glisse-les pour définir ton ordre d'initiative (mémorisé entre les sessions)
4. **F1** = perso suivant · **F2** = précédent · **Alt+1..8** = accès direct
5. Clique sur un perso dans la liste pour basculer sur sa fenêtre
6. Toutes les options dans le menu **⚙**

## ⚖️ Règles

Kali ne fait que **changer de fenêtre active** : aucune automatisation de gameplay,
aucune lecture des données du jeu. Conforme aux règles d'Ankama sur le multi-compte.
